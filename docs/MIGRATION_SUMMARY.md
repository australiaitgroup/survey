## 🚀 部署使用指南

### **生产部署命令**
```bash
# UAT环境
npm run deploy:uat

# 生产环境  
npm run deploy:prod
```

### **手动迁移选项**
```bash
# 分析迁移需求
npm run migrate:analyze

# 干运行测试
npm run migrate:dry-run  

# 交互式迁移
npm run migrate

# 自动迁移
npm run migrate:auto
```

### **关键环境变量**
```bash
MONGODB_URI=mongodb://...          # 必需
DEPLOY_ENV=uat|prod               # 部署环境
AUTO_MIGRATE_SURVEYS=true         # 自动迁移开关
MIGRATION_BACKUP=true             # 备份开关
```

```javascript

3. 全局数据: { slug }
```

## 📊 系统架构变化

### **之前 (单租户)**
```
┌─────────────────┐
│ Survey          │
├─────────────────┤
│ slug (unique)   │ ← 全局唯一约束
│ title           │
│ type            │
└─────────────────┘
```

### **之后 (多租户)**
```
┌─────────────────┐    ┌─────────────────┐
│ Survey          │    │ Company         │
├─────────────────┤    ├─────────────────┤
│ slug            │    │ slug (unique)   │
│ companyId   ────┼────→ _id            │
│ title           │    │ name            │
│ type            │    └─────────────────┘
└─────────────────┘
     │
     └─ 复合唯一: {slug, companyId}
```
